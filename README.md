## Reflection
The easiest part was setting up the HTML structure since I already knew it well.
The hardest part was remembering that JavaScript months are 0-based, so January is 0 not 1.
I learned the Date object is a built-in object you create with `new Date()` and it gives you
access to methods like `getMonth()` and `getFullYear()`. I learned the Number object has
static methods like `Number.isNaN()` that let you validate values without crashing the script.
Displaying results in the browser using `getElementById().innerHTML` was very satisfying to see working.
